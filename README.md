 Site Barbearia

Site institucional simples desenvolvido para uma barbearia, apresentando informações sobre serviços, preços e funcionamento.

O site possui um botão de agendamento que direciona o cliente diretamente para o WhatsApp da barbearia.

Tecnologias utilizadas

- HTML5
- CSS3
- SCSS
- JavaScript
- AOS Animation Library

 Responsividade

O site foi desenvolvido para funcionar em computadores e dispositivos móveis.
